## Universal Upload
*   **Format Support**: Wide range of supported formats including TrackAddict, RaceChrono, RaceBox, CirctuitStom, Lap Legend, Apex Pro, etc.
*   **Client-Side Processing**: Fast, secure upload with immediate feedback via client-side hashing and compression.

## Session Organization
*   **History Dashboard**: Personalized view of all uploaded sessions.
*   **Privacy Controls**: Toggle visibility between Public, Shared (Unlisted), and Private.
*   **Notes**: Integrated rich-text editor for driver notes per session.
