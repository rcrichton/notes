The core analysis engine designed for in-depth post-session review.

## Interactive Charts
*   **Speed Chart**: Visualize speed over distance or time.
*   **G-Force Charts**: Lateral and Longitudinal G-force analysis.
*   **Delta Chart**: Precise time-slip comparison against best laps or reference sessions.
*   **Toggle Modes**: Switch X-axis between **Distance** (track position) and **Time** (duration).

## Track Mapping
*   **Google Track Map**: Interactive map overlay showing the driving line.
*   **Data Layers**: Color-coded segments by lap

## Controls & Analysis Tools
*   **Lap Selector**: Quickly switch between laps to compare consistency.
*   **Kalman Filter**: Optional smart data smoothing to reduce sensor noise from mobile GPS.
*   **Playback**: Replay mechanism to watch the lap unfold in real-time or scrub through specific events.
*   **Timeline View**: A sequential event log showing Start/Finish crossings, sector times, and other key lap events.
